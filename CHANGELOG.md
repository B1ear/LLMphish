## [1.0.0] - 2025-12-03

### 新增

#### 核心功能
- ✨ 多模型融合检测系统
  - 简化 PhishMMF 模型（96% 准确率，AUC 0.98+）
  - IsolationForest 异常检测
  - 规则检测
  - LLM 语义检测（可选）

#### 简化 PhishMMF 模型
- 🎯 使用 35 个可提取特征重新训练
- 📊 RandomForest 和 XGBoost 双模型
- ⚡ 不依赖外部数据（URL OSINT、网站截图）
- 🚀 毫秒级响应时间

- 📊 实时检测结果展示

### 特性

#### 特征提取
- 文本特征（24维）
  - 主题分析：紧急程度、威胁性、诱惑性
  - 发件人分析：冒充检测、异常检测
  - 正文分析：关键词、URL、复杂度
- URL 特征（11维）
  - 域名分析：长度、结构、TLD
  - 路径分析：长度、参数
  - 可疑模式检测

#### 检测方法
- **简化 PhishMMF**: 基于 35 维特征的监督学习
- **IsolationForest**: 无监督异常检测
- **规则检测**: 关键词和模式匹配
- **LLM 检测**: 深度语义理解（可选）

#### 融合策略
- 加权投票
- 动态阈值
- 多级检测

### 已知问题

- 测试邮件样本有限
- LLM 检测需要 API Key
- 模型文件较大（~50MB）

### 安全

- 输入验证

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 更新类型

- **新增**: 新功能
- **变更**: 现有功能的变更
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: Bug 修复
- **安全**: 安全相关的修复
